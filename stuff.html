Stuff and Projects and Things!

Here you'll find a list of miscellaneous things I've made and decided to make public for others to enjoy. Right now there's not a lot here, though I hope to expand this section a bit in the future.

SuperslidesAreLegal A Hollow Knight CustomLogicInjector pack that adds acid bridge to logic with rightslash. See the CustomLogicInjector Github for instructions installing it!

Grubbynest A Hollow Knight CustomBGM pack that replaces some of the music in the game with grubs singing! Requires the CustomBGM mod. Again, you'll want to see the CustomBGM Github for instructions installing it. The download link is to a Google Drive folder, so you can choose which songs you want, or zip it and download the whole thing.
